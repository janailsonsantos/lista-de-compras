# Lista de Compras

Pequeno aplicativo de lista de compras que permite adicionar itens, marcar como concluído e remover itens da lista.

---

## 🚀 Tecnologias usadas

- **HTML** — estrutura do layout
- **CSS** — estilos e responsividade
- **JavaScript** — lógica para adicionar/remover itens dinamicamente

---

## 🧩 Estrutura do projeto

- `index.html` — página principal
- `script.js` — lógica de adicionamento/removal de itens
- `styles/` — estilos da aplicação
  - `global.css`
  - `main.css`
  - `index.css`

---

## ▶️ Como executar

1. Abra o arquivo `index.html` no navegador (duplo clique ou “Abrir com...”).
2. Digite um item no campo e clique em **Adicionar item**.
3. Clique no ícone de lixeira para remover um item.

---

## ✅ Funcionalidades

- Adicionar item na lista
- Remover item da lista (com notificação temporária)
- Layout adaptado para mobile (com ajustes de CSS)
